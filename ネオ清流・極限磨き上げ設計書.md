# 🌊 ネオ清流・極限磨き上げ設計書（UI/UX & 高密度レベルデザイン）

本作『鯉の滝登り（toryumon）』が持つ「和モダン極彩・超高速即死アクション」のポテンシャルを120%引き出し、プレイヤーが「タイトルを開いた瞬間」「死んだ瞬間」「UFOや釣り針に翻弄された瞬間」に脳汁（アドレナリン）が溢れ出す最高峰の手触りを実現するための、演出・設計・デザインの完全ガイドラインおよび実装設計書です。

## 1. 💀 一撃即死 ＆ 1秒リトライを「快感」に変えるUX

「死」はペナルティではなく、次への挑戦を熱狂的に煽る最上のスパイス（ジューシーなご褒美）として設計します。

### ① 即死の「大袈裟すぎる」ヒットストップ
* **演出法**: 障害物（岩や硬直針など）に衝突または即死ダメージを受けた瞬間、ゲーム全体のエンジン処理速度（`Engine.time_scale`）を **「0.08秒間だけ 0.05（ほぼ完全停止）」** に遷移させます。
* **心理的効果**: 衝突の「重み」が画面越しにプレイヤーの脳へズシンと伝わり、大破した実感が何倍にも増幅します。その後、通常速度（`1.0`）に戻った瞬間に溜め込まれたエネルギーを解放するように `JuicyDeathSplatter` が一気に爆散し、破裂の圧倒的カタルシスを生み出します。
* **Godot 4 実装スニペット（Player / GameController側）**:
  ```gdscript
  func trigger_juicy_death(hit_pos: Vector2) -> void:
      # 1. 劇的なヒットストップの開始
      Engine.time_scale = 0.05
      
      # 2. 0.08秒(実時間)後にスピード復帰＆パーティクル爆散
      get_tree().create_timer(0.08 * 0.05).timeout.connect(func():
          Engine.time_scale = 1.0
          spawn_death_splatter(hit_pos)
          trigger_death_monochrome_effect()
      )
  ```

### ② 死んだ瞬間の「色調反転」と「ズーム」
* **演出法**:
  1. **カメラズーム**: 衝突した瞬間に、`Camera2D` をプレイヤーの最終位置へ向けてスッと **1.2倍** ズームインさせます（Tweenで 0.1秒かけて `Vector2(1.2, 1.2)` へ補間）。
  2. **フラッシュ＆静寂**: 画面全体を「白練色（#F0F0F0）で1フレームだけフラッシュ」させた直後、全画面ポストプロセス（またはキャンバスシェーダー）で色調を **白黒（モノトーン）** に反転・彩度ゼロ化し、環境音やBGMをミュートした「1秒間の完全な静寂」を作ります。
* **画面との対比**: 普段の鮮烈な「白・青・緑・金」の極彩世界が、死んだ瞬間に「一瞬の完全な静寂とモノクロ」へ引き戻されることで、死の衝撃がアートワークとして美しく際立ちます。

### ③ Rキー・スペース連打を誘う「弾むリトライHUD」
* **演出法**: 衝突から **0.15秒後** に表示される「💥 衝突大破!! 💀」の文字やリトライ案内は、静止させず生命力を持たせます。
* **バウンド出現**: `JuicyButton` や `JuicyLabel` の設計を継承し、出現時にスケール `Vector2(0, 0)` から `Vector2(1.15, 1.15)` → `Vector2(1.0, 1.0)` へ、勢いよくぷにぷにとバウンド（Squash & Stretch）しながら飛び出させます。
* **心臓の鼓動（サイン波ループ）**: Rキーやスペースキーでの即時リトライを「早く押せ！」と急かすように、サイン波を用いて文字スケールを常時 `1.0` から `1.08` の間で脈動させます。
  ```gdscript
  # HUD側での鼓動アニメーション処理
  func _process(delta: float) -> void:
      if is_retry_hud_active:
          pulse_time += delta * 12.0 # 脈動スピード
          var scale_mod = 1.0 + (sin(pulse_time) * 0.04 + 0.04) # 1.0 〜 1.08
          retry_label.scale = Vector2(scale_mod, scale_mod)
  ```

## 2. 🎣 UFO混乱・釣り針連打を「パニックの楽しさ」に変えるディテール

UFOと釣り針は「単なる障害物」ではなく、プレイヤーの脳の処理能力を揺さぶり、最高にエキサイティングなパニック状態を作り出す一級品の主役ギミックです。

### ① 釣り針拘束時の「カメラのガタガタ激突シェイク」
* **演出法**:
  * 釣り針に捕まり下流へと激しく引きずり戻されている間、`Camera2D` を **上下左右に超高速でガタガタと震わせ続けます（連続Screen Shake）**。
  * プレイヤー頭上に展開する「ボタン連打で脱出!!」ゲージは、キーを連打するたびに **「パチン！バシャ！」** と白練色の水しぶき・泡パーティクルが跳ね上がり、ゲージが右へ弾むように伸びます。
* **焦りのタイム演出**: 拘束中は残り時間が通常の **2.5倍速** でゴリゴリ減少します。タイマー数字は赤色（#FF2A4B）に鋭く発光し、「プピピピピ！」と高速で上下バウンドさせることで、プレイヤーの焦燥感を最高潮へと導きます。

### ② UFOビーム突入時の「サイケデリック色収差」
* **演出法**:
  * ビームに触れて別レーンへ「神隠し（テレポート）」される瞬間、一瞬だけ画面全体を横方向に引き伸ばし、RGBのチャンネルをわずかにズラす **色収差（Chromatic Aberration）エフェクト** を発生させます。
  * ワープの瞬間、白練と神秘的な紫色（#8A2BE2 / 江戸紫）の「リング状波紋」がプレイヤー座標からブワッと急拡大して消えます。
* **左右反転のパニック表現**:
  * 操作が左右反転する **混乱（Confusion）の3秒間** は、頭上に現れる「❓左右反転❓」のテキストを和風の「お化け提灯」のようにユラユラと左右に揺動させます。
  * さらに、背景の滝スクロール速度を視覚的に10%遅らせるシマー効果（陽炎シェーダー）を適用し、「頭がグニャグニャと歪んでいる感覚」をダイレクトに視覚フィードバックします。

## 3. 🗺️ 通常泳ぎで美しく縫う「高密度・無ダッシュ」のパズル的レベルデザイン

ダッシュを「使わなくても進める」だけでなく、「通常泳ぎ（左右移動）だけでリズミカルに泳ぐと、パズルのピースが完璧に噛み合うように美しく障害物をすり抜けられる」ようにレベル・シーケンスを綿密に設計します。

### ① 3つの基本配置パターン

#### ターンA：「スラローム（蛇行）」
```text
[ 画面上部 ]
       ▲ (岩：右レーン)   <- 次にここを左へ避ける
   ▲ (岩：中央レーン)     <- 次にここを右へ避ける
▲ (岩：左レーン)          <- まずここを右へ避ける
[ プレイヤー ]
```
* **設計意図**: プレイヤーが `左` → `中` → `右` → `中` → `左` と、ちょうどメトロノームのリズム（約 **0.4秒間隔**）でキーをポン、ポン、ポンと叩くだけで、スタミナを消費するダッシュを一切使わずに流れるように突破できる至高の配置です。

#### ターンB：「フェイント（UFO誘導）」
```text
[ 画面上部 ]
▲ (岩：左レーン)  ▲ (岩：中央レーン)  <- 脱出先が完全に塞がれている！
             [ UFOビーム：右レーン ] <- あえてビームへ飛び込む！
[ プレイヤー ]
```
* **設計意図**: 一見すると右レーンのUFOビームが邪魔な罠に見えますが、あえてそこに突っ込むことで自動的に左や中央の安全圏へ「ワープ（神隠し）」し、目の前の巨大岩壁を裏口から華麗にすり抜けられるという、パズル攻略の深い快感を与えます。

#### ターンC：「サケピンボール」
```text
[ 画面上部 ]
▲ (岩：左レーン)                  ▲ (岩：右レーン)
              [ サケ：中央レーン ]
[ プレイヤー ]
```
* **設計意図**: 中央レーンを直進していると、突進してくる鮭に弾かれてプレイヤーは「左右どちらかに強制バウンド」させられます。しかし、バウンドした先には致命的な岩が待ち構えています。
* **攻略法**: プレイヤーは「右へ弾き飛ばされた瞬間に、即座に左キーを入力して中央へ復帰する」という、格闘ゲームのコンボ受け身のようなテクニカルでスタイリッシュな回避プレイを楽しめます。

## 4. 🔊 和モダン音響によるハプティクス（触覚）の統一

グラフィックの「白・青・緑」の清涼感と操作の「ハイスピードレスポンス」に対して、音響も格調高い和モダンテイストと物理的な手触り（ハプティクス）を完全にシンクロさせます。

| アクション / 状況 | 音響演出仕様 | 聴覚・触覚フィードバック |
| :--- | :--- | :--- |
| **決定・リトライ時** | 重厚でクッキリした **「和太鼓の ── ドンッ！」** | お腹の底に響く低音で、再挑戦への闘志を一瞬で点火する。 |
| **通常泳ぎ（レーン移動）** | 水面を軽やかに滑る **「プチッ」** または **「トッ」** | パーカッシブで極小の水滴音。連続操作時に心地良いクリック感を生む。 |
| **ダッシュ発動時** | 鋭い風切り音 **「ヒュッ！」** または刀の空切り音 | 空気を切り裂くような鋭い高音で、加速の疾走感を極限まで引き上げる。 |
| **即死爆散クラッシュ** | 陶器の粉砕音 ＋ 激しい水しぶき **「バシャーン！！」** | 陶器が粉々に砕け散る硬質な衝撃音と、豪快な水流音をミックスした一撃。 |
